# 🌐 Health Hack 2026 — VIT Bhopal × Johns Hopkins University

> **Improving Health Access for All**  
> A collaborative international health hackathon organized by **VIT Bhopal University** in association with **Johns Hopkins Whiting School of Engineering (USA)**.

---

## 🚀 Overview

**Health Hack 2026** is a premier hackathon event focused on fostering innovation in healthcare technology, accessibility, and global health solutions.  
This website is designed as a modern, **glassmorphic**, and **responsive single-page web application** to provide detailed information about the event, including registration, committees, timeline, tracks, and contact details.

---

## 🧩 Features

✅ **Fully Responsive UI** — Works on all devices (desktop, tablet, mobile)  
✅ **Glassmorphism Design** — Clean and modern design with neon & blur effects  
✅ **Dynamic Committees Tabs** — Organizing, Advisory, Faculty, and Industry sections  
✅ **Downloadable Brochure** — Direct PDF download from the Hero section  
✅ **Interactive Timeline** — Highlights key event milestones  
✅ **QR Code Integration** — Quick scan-to-register access  
✅ **Gallery Section** — Showcases previous hackathons and activities  
✅ **Collaboration Highlight** — Emphasizes partnership between VIT Bhopal and Johns Hopkins

## ⚙️ Setup & Usage

1. Clone or download this repository.
2. Place all images inside the `images/` folder.
3. Update registration links or QR code in `index.html` as needed.
4. Open `index.html` directly in your browser — no server required.
5. For hosting, upload to any static hosting service:
   - [Vercel](https://vercel.com/)
   - [GitHub Pages](https://pages.github.com/)
   - [Netlify](https://www.netlify.com/)

---

## 📦 Deployment Instructions

To deploy your website (Frontend Only):

### **Option 1: GitHub Pages**
1. Push your project to a GitHub repository.
2. Go to **Settings → Pages → Deploy from Branch → main**.
3. Your website will go live at `https://yourusername.github.io/HealthHack2026/`.

### **Option 2: Vercel**
1. Go to [vercel.com](https://vercel.com/).
2. Import your GitHub repo.
3. Click **Deploy** — your site will be hosted instantly with a custom URL.
